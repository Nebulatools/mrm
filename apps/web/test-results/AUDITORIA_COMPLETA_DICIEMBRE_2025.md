## 🐛 PROBLEMAS DETECTADOS Y SOLUCIONES

### Problema 1: Botón "Generar" no visible en tab Rotación
**Severidad:** BAJA (UI)
**Descripción:** Componente de narrativa fuera de viewport en algunos tabs
**Impacto:** Usuario debe hacer scroll manual
**Solución:**
```typescript
// En rotacion-tab.tsx o smart-narrative.tsx
useEffect(() => {
  // Scroll al componente cuando se monta
  const narrativeElement = ref.current;
  if (narrativeElement) {
    narrativeElement.scrollIntoView({ behavior: 'smooth', block: 'nearest' });
  }
}, []);
```

### Problema 2: No valida población filtrada = 0
**Severidad:** MEDIA (Funcional)
**Descripción:** Permite generar narrativa con 0 empleados
**Solución:** YA documentada en reporte anterior

### Problema 3: Selectores de filtros complejos para testing E2E
**Severidad:** BAJA (Testing)
**Descripción:** Dropdowns con checkboxes dificultan automatización
**Solución:**
```tsx
// Agregar data-testid a elementos clave
<button data-testid="filter-year-dropdown">
<checkbox data-testid="filter-year-2025">
<button data-testid="generate-narrative">
```
